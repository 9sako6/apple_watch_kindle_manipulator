# Apple Watch Kindle Remote

Apple Watchのメディア操作を、iPad Safari上のKindle for Web / Kindle Cloud Readerのページ送りに変換するUserscripts用ユーザースクリプトです。

## 開発

```sh
npm install
npm run check
npm run build
```

実装は `src/userscript.ts` に集約しています。生成物は `dist/kindle-remote.user.js` の1ファイルです。

インストール方法と使い方は [`docs/README.md`](docs/README.md) を参照してください。
